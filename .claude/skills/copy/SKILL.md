# Skill: /copy

Gera copy para qualquer contexto do ligcentro a partir de um briefing estruturado — da headline da landing page ao microcopy de UI, passando por e-mails, posts e CTAs.

## Quando usar

- Quando o copywriter for invocado para escrever copy novo.
- Quando digital-marketer precisar de texto para campanha.
- Quando social-media-manager precisar de caption de post.
- Quando frontend-developer precisar de microcopy para componente novo.

## Briefing mínimo necessário

Antes de escrever, confirmar que você tem:
1. **Contexto**: onde vai o texto? (landing page / UI / e-mail / Instagram / LinkedIn)
2. **Objetivo**: o que o texto deve fazer? (converter cadastro / educar / engajar / reter)
3. **Persona**: quem vai ler? (Camila / Rafael / Dona Márcia / Thiago — ver `docs/marketing/personas.md`)
4. **Mensagem-chave**: qual das M1–M6 este texto reforça? (ver `docs/marketing/key-messages.md`)
5. **Keyword** (se SEO): qual keyword primária integrar naturalmente?
6. **Restrição de tamanho**: limite de caracteres ou palavras?

Se algum item estiver faltando, perguntar antes de escrever.

## Passos

1. **Leia** `docs/marketing/brand-voice.md`, `docs/marketing/personas.md` e `docs/marketing/key-messages.md`.

2. **Escreva** 2–3 variações quando o contexto for landing page ou ad (para A/B test).
   Para microcopy de UI ou posts: 1 versão com nota de variação.

3. **Auto-revisar** contra `/content-review` antes de entregar:
   - Direto, honesto, benefício-primeiro, sem jargão, em pt-BR nativo.
   - Mensagem-chave presente sem forçar.
   - Nenhum dark pattern.

4. **Para microcopy de UI**: entregar em pt-BR E en-US simultaneamente.

5. **Entregar** copy finalizado + breve nota sobre decisões de copy relevantes.

## Templates por formato

### Hero da landing page
```
[Headline — até 8 palavras, benefício principal]
[Subheadline — 1–2 linhas, expande o headline com especificidade]
[CTA — verbo de ação + benefício — até 5 palavras]
```

### Microcopy de UI (empty state)
```
[Ícone contextual]
[Título — o que está vazio]
[Corpo — 1 frase motivando a ação, sem pressão]
[CTA — verbo de ação]
```

### E-mail (assunto + preview)
```
Assunto: [< 50 chars — curioso ou direto; evitar clickbait]
Preview text: [< 90 chars — complementa o assunto, não repete]
```

### Instagram caption
```
[Hook — 1ª linha visível antes do "ver mais" — deve parar o scroll]
[Corpo — 2–4 linhas, dica/insight/pergunta]
[CTA — link na bio / salva / comenta]
[Hashtags — 5–10, mistura geral + nicho]
```

### LinkedIn post
```
[Hook — 1ª linha que para o scroll]
[Desenvolvimento — 3–5 parágrafos curtos, cada um com ideia própria]
[Fechamento — insight ou pergunta para engajamento]
[CTA sutil — não forçado]
```
