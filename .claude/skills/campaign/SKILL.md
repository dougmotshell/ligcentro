# Skill: /campaign

Planeja uma campanha de marketing do ligcentro de ponta a ponta — do objetivo ao calendário de execução — com briefings para cada agente envolvido.

## Quando usar

- Quando o digital-marketer for planejar uma campanha de aquisição, lançamento ou retenção.
- Para o lançamento do MVP (Fase 4).
- Para campanhas sazonais (Dia do Creator, Black Friday, etc.).
- Para campanhas de captura de ex-usuários do Bento.

## Passos

1. **Confirmar objetivo da campanha** (um só):
   - Aquisição (novos cadastros)
   - Ativação (cadastrados que não publicaram perfil)
   - Retenção (usuários que ficaram inativos)
   - Conversão (free → Pro)
   - Awareness (brand awareness pré-lançamento)

2. **Definir a campanha**:
   ```
   Nome: [nome interno da campanha]
   Objetivo: [aquisição / ativação / retenção / conversão / awareness]
   Persona-alvo: [Camila / Rafael / Dona Márcia / Thiago]
   Mensagem-chave: [M1–M6 de docs/marketing/key-messages.md]
   Período: [data início → data fim]
   Canais: [Instagram / TikTok / LinkedIn / E-mail / SEO / todos]
   Métrica de sucesso: [X cadastros / Y aberturas de e-mail / Z cliques]
   ```

3. **Gerar briefings por agente**:

   **→ copywriter**: headline + body + CTAs por canal
   **→ content-marketer**: artigos ou conteúdo de suporte necessário
   **→ social-media-manager**: posts por plataforma, calendário social
   **→ seo-specialist**: keywords a cobrir, páginas a otimizar
   **→ sales-specialist** (se conversão): flow de upgrade a criar/otimizar

4. **Calendário de execução**:
   ```
   Semana -2: [preparação — copy, assets, landing page]
   Semana -1: [revisão, agendamento]
   Dia 0: [lançamento]
   Semana +1: [monitoramento e ajuste]
   Semana +2: [relatório final]
   ```

5. **Checklist pré-lançamento**:
   - [ ] Copy revisado pelo copywriter contra `/content-review`
   - [ ] Landing page / página de destino funcionando
   - [ ] UTM links gerados para cada canal
   - [ ] Nenhum dark pattern na campanha
   - [ ] LGPD: nenhuma coleta de dado de visitante sem consentimento
   - [ ] Métricas de baseline registradas (para comparar depois)

## Output esperado

Documento markdown com: briefing da campanha + briefings por agente + calendário de execução + checklist.
